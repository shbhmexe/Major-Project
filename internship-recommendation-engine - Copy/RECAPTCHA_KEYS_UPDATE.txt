# reCAPTCHA Keys Update Template

## एक बार आपको keys मिल जाएं, तो .env में ये lines replace करें:

FROM (Current):
NEXT_PUBLIC_RECAPTCHA_SITE_KEY=your-recaptcha-site-key
RECAPTCHA_SECRET_KEY=your-recaptcha-secret-key

TO (Your actual keys):
NEXT_PUBLIC_RECAPTCHA_SITE_KEY=6LcXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
RECAPTCHA_SECRET_KEY=6LcXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX

## Keys के बाद app test करने के लिए:

```bash
npm run dev
```

## Test Flow:
1. http://localhost:3000 पर जाएं  
2. Phone number registration try करें
3. reCAPTCHA challenge appear होना चाहिए
4. OTP send होना चाहिए without errors
5. Dashboard पर successful redirect होना चाहिए

## Expected Results:
- ✅ No reCAPTCHA errors in console
- ✅ Phone OTP working
- ✅ Smooth navigation after auth
- ✅ Dashboard showing after successful login

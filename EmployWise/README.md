## Getting Started

1. Clone the repository
2. Install dependencies
   ```
   npm install
   ```
3. Run the development server
   ```
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:5173`

## Production Build

To create a production build, run:
```
npm run build
```
